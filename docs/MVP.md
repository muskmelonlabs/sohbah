\# Sohbah - MVP Scope



\---



\## 1. Objective



Build a simple, working version of Sohbah that validates:



Users can discover mentors and request sessions easily.



\---



\## 2. Features Included



\### 2.1 Mentor Listing

\- Static mentor cards

\- Name, topics, short bio



\---



\### 2.2 Mentor Profile

\- Detailed mentor info

\- CTA to book session



\---



\### 2.3 Booking Flow

\- Form:

&#x20; - Name

&#x20; - Topic

&#x20; - Preferred time



\---



\## 3. Data



\- mentors.json (static)

\- Booking stored locally or logged



\---



\## 4. Tech



\- HTML

\- CSS

\- JavaScript

\- No backend



\---



\## 5. UX Scope



\- Clean UI

\- Mobile responsive

\- Minimal flow



\---



\## 6. Definition of Done



User can:

\- Browse mentors

\- View profile

\- Submit booking form



\---



\## 7. Not Included



\- Login

\- Payments

\- Chat

\- Calendar

\- Video calls
